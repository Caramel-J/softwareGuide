## 特性

* **操作方便** ： 只需动动鼠标，无需过多的数据录入等复杂操作
* **安全** ：权限精细，保证数据的安全性
* **追溯** ： 通过质量模块中的功能更方便的了解产品的去向
* **便捷** ： 方便接入第三方物流，您无需把精力花费在仓库管理上
* **个性化** ： 支持个性化设置，头像更换、待办事宜等
